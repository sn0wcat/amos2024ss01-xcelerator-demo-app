Welcome to the
Siemens Xcelerator Demo App

With this app, you can:

  • - Monitor your water tanks and associated devices like pumps
  • - View the status of tanks and devices in an interactive user interface
  • - Create cases for servicing your water tanks

This application was done by students of FAU Erlangen, TU Berlin and FU Berlin as a part of
the AMOS Project (Agile Methods in the Open Source)

The source code for the application can be found on GitHub and
the architecture description is available in the GitHub wiki..
Here you can find more informations about Siemens Xcelerator.

The documentation about Siemens APIs is available on the Siemens Xcelerator Developer Portal.

The application frontend was built using Siemens Industrial Experience - an open-source
design system for designers and developers.
